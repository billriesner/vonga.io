## üé® Additional Generation Tips for RunwayML

### General Settings:
- **Model:** Gen-3 Alpha Turbo (faster, good quality)
- **Duration:** 5-10 seconds per clip
- **Resolution:** 1280x768 (upscale to 1080p in post)
- **Seed:** Save seed for consistent style across shots
- **Style Reference:** Use your first successful generation as reference for subsequent shots

### Prompt Modifiers to Add:
For ALL prompts, you can append:
```
, professional commercial quality, 8K, sharp focus, perfect lighting, 
award-winning cinematography, ultra-realistic
```

### For UI/Dashboard Shots:
Since RunwayML struggles with precise UI, consider:
1. **Hybrid approach:** Design UI in Figma ‚Üí Export ‚Üí RunwayML image-to-video
2. **Focus on backgrounds:** Generate the environment, add UI in After Effects
3. **Abstract data viz:** Generate abstract data visualizations, composite later

### Color Grading:
- **Shadows:** Slightly lifted (not pure black)
- **Highlights:** Preserve aqua (#33BECC) and coral (#FF6B6B)
- **Overall:** Slightly desaturated for professional B2B feel
- **Consistency:** Match all clips to same color space
