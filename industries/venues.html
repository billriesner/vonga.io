Works With Any Tenant, Any Event

How It Works

NFC touchpoints are installed once. The content behind each tap changes per event:

  • NHL game: Team content, team sponsors, team loyalty program
  • Concert: Artist merch, tour offers, streaming unlocks
  • Conference: Attendee networking, sponsor booths, session check-ins

What the Venue Always Gets

Regardless of event type or tenant configuration, the venue captures:

  • Event attendance pattern
  • Venue location of interaction
  • Cross-event profile enrichment
